## About

**Weather App** lets users search for current weather conditions in any city around the world. It displays the temperature, weather icon, location, and current date and time. Users can also switch between Celsius and Fahrenheit units using a dropdown in the header.

---

## Features

- 🔍 **City search** — search for any city's current weather
- 🌡️ **Unit toggle** — switch between Celsius (°C) and Fahrenheit (°F)
- 🗓️ **Date & time display** — shows the current date and local time
- 🌤️ **Weather icon** — dynamic icon reflecting current weather conditions
- 📍 **Location display** — shows city and country name
- 🌐 **Live weather data** — fetched from a weather API in real time

## Technologies Used

- **HTML5** — semantic page structure
- **CSS3** — layout and responsive styling
- **JavaScript (ES6+)** — DOM manipulation, Fetch API, event handling
- **Google Fonts** — [Bricolage Grotesque](https://fonts.google.com/specimen/Bricolage+Grotesque) & [DM Sans](https://fonts.google.com/specimen/DM+Sans)
- **Weather API** — live weather data (e.g. [OpenWeatherMap](https://openweathermap.org/api))
